# TheContenator

Creator research and content intelligence platform. Discover top creators in any niche, analyze video performance, extract scripts, and download content across TikTok, YouTube, and Instagram.

## Features

- **Niche Explorer** — Search a niche, get ranked list of top creators
- **Account Browser** — Browse any creator's videos sorted by views, comments, engagement, date
- **Video Analyzer** — Full stats breakdown for any video
- **Script Extractor** — Transcribe video audio into a readable script
- **Downloader** — Download videos from any supported platform

## Quick Start

```bash
git clone https://github.com/monstersebas1/TheContenator.git
cd TheContenator
python -m venv venv && source venv/bin/activate
pip install -r requirements.txt
cp config/config.example.json config.json
python src/main.py
```

Requires Python 3.12+.
